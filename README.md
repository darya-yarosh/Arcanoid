# Arcanoid

* Технические требования:
1) Язык: JavaScript;
2) Библиотеки: можно любые;
3) Игровые движки: можно любые.

* Обязательно:
1) Дизайн, рассчитанный на массового игрока;
2) Работоспособность на:
- Мобильных устройствах;
- Десктопных браузерах.
3) Геймплей:
- не появляется желание закрыть игру после первой минуты;
- приятное управление;
- анимации;
- звук (при желании).
4) Смена уровней;
5) Отсутствие багов;
6) Производительность;
7) Чистый код.

* По желанию:
1) Выпадающие бонусы;
2) Набор очков;
3) Адаптивность под разные размеры/разрешения девайсов;
4) Оптимизированный размер билда;
5) Остальное на усмотрение.
